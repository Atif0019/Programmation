Qu'est-ce que JavaScript ?

JavaScript (JS) est un langage de programmation informatique utilisé pour rendre les sites Web et les applications dynamiques et interactifs. C'est unique car il peut s'exécuter directement dans votre navigateur, pas seulement sur un serveur. Avec le langage de balisage hypertexte (HTML) et les feuilles de style en cascade (CSS), JavaScript est l'un des langages de programmation les plus couramment utilisés sur Internet.

Histoire de JavaScript

Brendan Eich a développé JavaScript en 1995 alors qu'il travaillait pour Netscape. Netscape cherchait à développer un langage de script qui pourrait contribuer à rendre les premières versions du World Wide Web plus dynamiques. JavaScript a rapidement gagné en popularité à mesure que les développeurs ont réalisé sa capacité à ajouter du dynamisme et de l'interactivité aux pages Web. En 1996, Netscape a soumis JavaScript à l'Association européenne des fabricants d'ordinateurs (ECMA International), une organisation qui vise à normaliser l'utilisation des systèmes informatiques.
