Qu’est-ce que le HTML ?

D'accord, c'est donc la seule théorie obligatoire. Pour commencer à écrire du HTML, il est utile de savoir ce que vous écrivez. HTML est le langage dans lequel la plupart des sites Web sont rédigés . HTML est utilisé pour créer des pages et les rendre fonctionnelles. Le code utilisé pour les rendre visuellement attrayants est connu sous le nom de CSS et nous y reviendrons dans un didacticiel ultérieur. Pour l’instant, nous allons nous concentrer sur vous apprendre à construire plutôt qu’à concevoir.

L'histoire du HTML

HTML a été créé pour la première fois par Tim Berners-Lee, Robert Cailliau et d'autres à partir de 1989 . Cela signifie Hyper Text Markup Language. L'hypertexte signifie que le document contient des liens qui permettent au lecteur d'accéder à d'autres endroits du document ou à un autre document. La dernière version est connue sous le nom de HTML5 . Un langage de balisage est un moyen par lequel les ordinateurs communiquent entre eux pour contrôler la manière dont le texte est traité et présenté. Pour ce faire, HTML utilise deux choses : les balises et les attributs .
